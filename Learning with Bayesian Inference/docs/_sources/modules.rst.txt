API
***

.. contents::

The API describes all functionalities implemented for this exercise. 
Of particular interest is the :mod:`main` script, which can be used to validate an implementation.

The module :mod:`lib.utils` contains the missing functionalities that have to be implemented. 
You have to **precisely follow the interface as described in section** :mod:`lib.utils` when implementing the missing functionalities!
